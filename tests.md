# Тестирование приложения Book Tracker

**Проект:** Book Tracker  
**Автор:** Камалетдинова Азалия Айратовна  
**Назначение тестирования:** проверка корректной работы GUI-приложения для управления списком прочитанных книг.

---

## 1. Проверяемые функции

В ходе тестирования проверялись следующие возможности:

- запуск графического приложения;
- добавление новой книги;
- проверка корректности ввода;
- редактирование записи;
- удаление записи;
- фильтрация книг по жанру;
- фильтрация книг по количеству страниц;
- сохранение данных в JSON;
- загрузка данных из JSON;
- отображение статистики по книгам и страницам.

## 5. Проверка JSON

Пример структуры файла `books.json`:

```json
[
  {
    "title": "Мастер и Маргарита",
    "author": "Михаил Булгаков",
    "genre": "Роман",
    "pages": 480
  }
]
```

Проверено:

- данные сохраняются в JSON;
- русские символы отображаются корректно;
- после перезапуска книги загружаются обратно;
- после удаления записи JSON обновляется;
- статистика рассчитывается по текущим данным.
