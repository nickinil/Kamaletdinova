# Book Tracker - Трекер прочитанных книг

**Автор:** Камалетдинова Азалия Айратовна

## Описание программы

Book Tracker - это десктопное приложение с графическим интерфейсом для управления списком прочитанных книг. Программа позволяет добавлять, редактировать, удалять книги, фильтровать их по жанру и количеству страниц, а также просматривать статистику.

### Основные возможности

- ✅ Добавление книг с информацией: название, автор, жанр, количество страниц
- ✅ Редактирование и удаление записей
- ✅ Фильтрация книг по жанру
- ✅ Фильтрация книг по количеству страниц (больше указанного значения)
- ✅ Автоматическое сохранение данных в JSON-файл
- ✅ Валидация вводимых данных
- ✅ Статистика: общее количество книг и страниц

## Требования

- Python 3.6 или выше
- Стандартные библиотеки Python (tkinter, json, os)
